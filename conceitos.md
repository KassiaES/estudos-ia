# Conceitos de IA Generativa e Prompting

## Prompt

Um **prompt** é uma entrada fornecida a um modelo de IA Generativa (GenAI) com o objetivo de orientar a saída produzida por ele. Pode ser uma pergunta, instrução, trecho de texto ou qualquer outro tipo de entrada que direcione o comportamento do modelo.

## Prompting

**Prompting** é o processo de fornecer um prompt a uma GenAI para que ela gere uma resposta. É a interação básica entre o usuário e o modelo: o usuário escreve um prompt e o modelo retorna uma saída com base nele.

## Prompt Chaining

**Prompt chaining** (encadeamento de prompts) consiste em concatenar dois ou mais prompts de forma sequencial, onde a saída de um prompt se torna a entrada do seguinte. Essa técnica é útil para dividir tarefas complexas em etapas menores e mais controladas.

## Técnicas de Prompting

**Técnicas de prompting** são modelos (padrões) que descrevem como estruturar um prompt ou como sequenciar múltiplos prompts para obter melhores resultados de um modelo de GenAI. Também são chamadas de **padrões de prompt** (*prompt patterns*). Exemplos comuns incluem *zero-shot*, *few-shot*, *chain-of-thought*, entre outros.

## Engenharia de Prompt

**Engenharia de prompt** (*prompt engineering*) é o processo iterativo de desenvolvimento de um prompt, no qual o autor modifica ou altera a técnica de *prompting* usada com o objetivo de refinar e melhorar progressivamente as respostas geradas pelo modelo.
